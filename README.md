# 🏠 D4 Inmobiliaria

Proyecto desarrollado para el **Desafío 4 - Inmobiliaria** de Desafío Latam.

El sitio muestra propiedades en venta y arriendo creadas dinámicamente con JavaScript a partir de arreglos de objetos.

## ✨ Características

- Página principal con 3 propiedades en venta y 3 en arriendo.
- Páginas separadas para ver todas las propiedades.
- Tarjetas generadas con ciclos, interpolación e `innerHTML`.
- Condicionales para indicar si se permite fumar o tener mascotas.
- Diseño adaptable en colores negro, dorado y azul.
- Navegación entre todas las páginas.

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript

├── propiedades_venta.html

## 🚀 Cómo visualizarlo

Abre el archivo `index.html` en tu navegador para ver la página principal.

Desde el menú o los botones puedes visitar las páginas de propiedades en venta y arriendo.
